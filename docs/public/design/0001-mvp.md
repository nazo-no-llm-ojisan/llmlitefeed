# LLM LiteFeed MVP

## Problem
既存のAIニュース収集ツールは巨大で、導入・設定・運用が重い。

## Goal
バイナリを起動し、LLMと興味を指定すると、
ローカルのニュースサイトが数秒で表示される。

## Non-goals
- 一般Webスクレイピング
- X連携
- MCP
- RAG
- プラグイン
- 自動公開
- マルチユーザー

## First vertical slice
1. 単一バイナリを起動
2. localhostで管理画面を開く
3. RSSを1件登録
4. 記事を取得
5. SQLiteへ保存
6. 一覧表示

## Invariants
- localhost以外へbindしない
- テストは実データへ触れない
- APIキーをSQLiteへ保存しない
- すべての外部URLを共通fetcherへ通す
- 外部入力をinnerHTMLへ入れない

## Done when
- Windows/macOS/Linuxでビルド可能
- 空のユーザーデータ領域から起動可能
- fixture RSSを取得して画面表示可能
- 全テストが一時SQLiteだけを使用
